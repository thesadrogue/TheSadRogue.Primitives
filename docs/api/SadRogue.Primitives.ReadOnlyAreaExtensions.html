Type Description
ReadOnlyAreaPositionsEnumerable

A custom enumerator that iterates over the positions in the area in the most efficient manner possible.
Remarks

The enumerator returned will use the area's indexer to iterate over the positions (like you might a list), if the area's UseIndexEnumeration is true. Otherwise, it uses the typical IEnumerator implementation for that area.

This may be significantly faster than the typical IEnumerable/IEnumerator usage for implementations which have UseIndexEnumeration set to true; however it won't have much benefit otherwise.

Remarks

Typically, you will want to use AdjacencyRule.EightWay as the rule; however AdjacencyRule.Cardinals is faster if you don't want border cells adjacent to a wall ONLY diagonally to be considered border cells.

Using AdjacencyRule.Cardinals, if "." and "x" are cells within the area and "#" are cells that are not within, X will NOT be considered a border:
