Class RectanglePositionsEnumerable

A custom enumerator used to iterate over all positions within a rectangle with a foreach loop efficiently.

public sealed class RectanglePositionsEnumerable : ValueType
Remarks

This type is a struct, and as such is much more efficient than using the otherwise equivalent type of IEnumerable<Point> with "yield return". The type does contain a function ToEnumerable() which creates an IEnumerable<Point>, which can be convenient for allowing the returned positions to be used with LINQ; however using this function is not recommended in situations where runtime performance is a primary concern.
